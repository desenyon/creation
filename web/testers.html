Early tester program

Help test the build loop that keeps going after agents stop.

Creation is for indie devs and solo founders with real repos. Give it a task, let it build, run tests, open the app in a browser, patch failures, and leave proof of what happened.

View GitHub →

What we need

  • Try Creation on an existing repo or a new throwaway product.
  • Tell us where the loop stopped, repeated itself, or failed to verify the app.
  • Share terminal output, screenshots, or a run receipt if you have them.

Feedback portal

Join as a tester or send a run report

What makes a useful test

Existing repo Run it where you already have tests, scripts, routes, and weird setup.
Broken path Ask Creation to fix something that currently fails instead of a blank demo.
Browser route Tell us the URL or user flow it should verify in the browser.
Stop point The best report is where Creation stopped, looped badly, or missed an obvious patch. 
creation run --workdir . --idea "Fix the failing signup flow and verify it in the browser"
