Pricing

A free, open-source AI coding tool

Creation is a free AI coding tool — open-source under the MIT license, with no subscription and no seat fee. It's an agentic coding tool that runs on your own machine, so the only costs are the third-party services you choose to plug in.

Is it really free?

Yes. There is no Creation plan to buy and nothing to upgrade. You install it with one command and run it locally. Because Creation orchestrates coding agents rather than reselling a model, you bring your own — and many of the 43 supported coding-agent CLIs run on a login you already have, with no extra API key.

What "free" actually means here

  • No Creation fee — the tool itself is free and open-source.
  • Your keys, your bill — if you connect a paid model or API, that provider bills you directly. Creation never marks it up.
  • Free path exists — pair it with coding agents that use your existing CLI login and you can run an agentic coding loop at no added cost.
  • Try free — creation run --demo runs the full pipeline in demo mode.
